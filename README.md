
# Boodschappen Doen

This project is about creating a grocery list based on selected dishes.
Pick one or more dishes in the UI, and the app combines their ingredients into a practical shopping list!

# Installation pre-requisites

Please use Node 18 LTS (Long Term Support version).

# Installing the Angular CLI

Install the Angular CLI globally with:

    npm install -g @angular/cli

# To run the Development UI Server

Install dependencies first:

    npm install

Run the frontend app with:

    npm start

The application is available at: [http://localhost:4200](http://localhost:4200)
